About

the project was developed by Rotem Sharon a 28 year old web developer as a final project for his full stack studies. The project includes front end (client side) and back end (server side), and connected to a database

About the project

front end - client side

To enhance the visual design of the website, I used CSS and Bootstrap. With CSS, I styled the HTML pages in a modern way, and I utilized Bootstrap for buttons, menus, tables, and the overall layout of the site. I also incorporated JavaScript to add animations to the UI components, providing a smooth and intuitive user experience.

back end - server side

The back end of the site is built using Node.js and Express.js, providing a robust and efficient server-side framework. I utilized Express Router to manage the routing of different endpoints, ensuring a clear and organized structure for the application's various routes. The site is developed following the MVC (Model-View-Controller) architecture, which allows for a clean separation of concerns. This architecture facilitates easy maintenance and scalability, as the data management (Model), user interface (View), and application logic (Controller) are all handled independently yet cohesively.

data base

All data for the site is stored in a MySQL database, ensuring reliable and structured data management. I connected to the database using libraries such as Knex, MySQL, and MySQL2, which provided a seamless interface for querying and managing the database. Knex was particularly useful for building and executing SQL queries in a flexible and efficient manner, while MySQL and MySQL2 facilitated the connection and interaction with the database, ensuring smooth data retrieval and storage operations throughout the application.
