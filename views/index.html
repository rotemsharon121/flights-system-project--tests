welcome to Blue sky

The main goal of the project is to create a system that will assist in managing all processes related to the aviation traffic system in a centralized and efficient way. This includes the management of airlines companies, flights details, tickets management, and customers information.

Summary of the project

flights management system project

With the tools that the project will provide you, you will be able to apply actions on the following sections:

Airline companies

Flights

Tickets

Customers

Countries

Admins

the actions you can apply:

  • look for a specific data
  • add data
  • update data
  • delete data

All data is stored in a mysql database

The client side is created with

Html, Css, Javascript, bootstrap and bootstrap-icons

The server side is created with

node.js, express.js, express.router

I connected to the database using libraries

knex, mysql, mysql2

To encrypt the passwords i used a library bcrypt

The site was built in architecture mvc- model, view, controller
